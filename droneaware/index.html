About DroneAware
DroneAware is a drone information app that helps you understand drone models, their specs, Remote ID support, and the regulations that apply in your country.

This app does not perform any drone detection or signal scanning. It is purely an information and reference tool.

Built with ❤️ using HTML, CSS & JavaScript.
